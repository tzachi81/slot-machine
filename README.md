# Slot-Machine
slot-machine game jQuery + php


# Run
## Server
From [ROOT] folder, run:
(no need to cd into /server folder)
- `php -S localhost:8000 -t server/public`

## Client
From [ROOT]/client folder, run:

### Development
  - `npm install`
  - `npm run dev`

### Production
  - `npm run build`
  - `npm run start`
